# 12week_Docker_startspark

## Spark 학습 프로젝트

이 저장소는 Apache Spark와 Docker를 활용한 데이터 처리 학습을 위한 프로젝트입니다.

### 프로젝트 구조
```
learning_spark_data/
├── notebooks/          # Jupyter 노트북 파일들
├── data/              # 데이터 파일들
├── scripts/           # Spark 스크립트들
├── docker/            # Docker 관련 파일들
└── docs/              # 문서들
```

### 기술 스택
- Apache Spark
- Docker
- Python
- Jupyter Notebook
- MySQL (Docker)

### 시작하기
1. Docker 설치
2. Spark 컨테이너 실행
3. Jupyter 노트북 접속

### 학습 목표
- Spark RDD, DataFrame, Dataset 이해
- Spark SQL 활용
- Docker 환경에서의 Spark 실행
- 실시간 데이터 처리
